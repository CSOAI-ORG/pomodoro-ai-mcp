# Pomodoro AI MCP

Manage Pomodoro focus sessions with configurable work and break intervals and session counters.

## Tools

- `start_pomodoro` — Start a Pomodoro work session.
- `check_status` — Check remaining time of the active session.
- `get_stats` — Get completed Pomodoro session statistics.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m pomodoro-ai-mcp.server
```
